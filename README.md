### Getting started

  1. Clone
  ```
   git clone https://github.com/zilurrane/movie-mate-api.git
  ```

  2. Install packages
  ```
   npm install
  ```
  
  3. Update .env file
  ```
  PORT=9000
  MONGO_CON_STRING=<your_mongodb_connection_string>
  JWT_SECRETE=<jwt_secret>
  ```
  
  4. Build
  ```
   npm run build
  ```
  
  5. Start
  ```
   npm start
  ```
  
  6. Development Start with auto-reload on changes
  ```
   npm run dev
  ```
  
#  Happy Coding!
